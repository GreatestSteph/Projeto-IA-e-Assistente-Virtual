# NicoAI

O NicoAI é um agente de IA que acompanha você na sua jornada de estudos sobre o mercado financeiro, atuando como um tutor virtual personalizado. 
Através da leitura inteligente de dados locais como o histórico de transações e perfis de investidor, ele contextualiza conceitos complexos e simula cenários econômicos de forma interativa, tornando o aprendizado sobre finanças pessoais acessível, seguro e totalmente prático.
